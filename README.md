# SoapLogMonitor — Frontend

React + TypeScript + Vite dashboard for monitoring SOAP logs.

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- Backend API running at `http://localhost:5234`

## Install dependencies

```bash
npm install
```

## Run

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.
